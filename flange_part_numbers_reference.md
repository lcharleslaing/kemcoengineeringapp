# Top Flange Part Numbers Reference

## Overview
This document lists all part number assignments for Top Flanges #1 through #5 based on:
- Flange Size (1.5", 2", 2.5", 3", 4", 6", 8", 10", 12")
- Material: SS316
- SS316_FLANGES flag (True/False)
- Food Grade (currently commented out in code)

---

## Part Number Logic Summary

### Base Part Numbers (Non-food grade, SS316_FLANGES = False)
- **1033918-13** = 1-1/2" SS316
- **1033918-01** = 2" SS316
- **1033918-02** = 2-1/2" SS316
- **1033918-03** = 3" SS316
- **1033918-04** = 4" SS316
- **1033918-06** = 6" SS316
- **1033918-08** = 8" SS316
- **1033918-10** = 10" SS316
- **1033918-12** = 12" SS316

### SS316 Flanges (SS316_FLANGES = True)
- Same base numbers with "SS" suffix
- Example: **1033918-13SS** = 1-1/2" SS316 (with SS316_FLANGES flag)

### Food Grade (Currently Commented Out)
- Base: **1051839-XX** series
- Same size suffixes as above

## Decision Tree

```
Is MATERIAL = "SS316"?
├─ YES
│  ├─ Is SS316_FLANGES = True?
│  │  ├─ YES → Use part number with "SS" suffix
│  │  └─ NO → Use standard part number
│  └─ Is FoodApplication = True? (Currently disabled)
│     └─ YES → Use 1051839-XX series
└─ NO → No part number assigned
```
