# RFSO FLANGE 150LB Parameters Reference

## Overview
This document lists all parameter values for RFSO FLANGE 150LB:1 based on FlangeSize. These are dimensional parameters used in Inventor iLogic to configure the flange geometry.

**Component:** RFSO FLANGE 150LB:1

**Available Sizes:** 1.5", 2", 3", 4", 6", 8", 10", 12"

---

## Parameter Definitions

| Parameter | Description | Unit |
|-----------|-------------|------|
| O | Outside Diameter | in |
| C1 | Flange Thickness | in |

| Y | Hub Length | in |
| B | Bore Diameter | in |
| BC | Bolt Circle Diameter | in |
| DH | Bolt Hole Diameter | in |
| NU | Number of Bolts | ul (unitless) |

## Notes

- **NU (Number of Bolts):** Changes from 4 to 8 at 4" size, then to 12 at 10" size
- **DH (Bolt Hole Diameter):** Changes from 0.625" to 0.75" at 2" size, then to 0.875" at 6" size, then to 1.00" at 10" size
- **C1 (Flange Thickness):** Generally increases with size, but remains constant at 0.88" for both 3" and 4" sizes
- **Y (Hub Length):** Increases progressively with flange size
- **BC (Bolt Circle):** Increases proportionally with outside diameter
