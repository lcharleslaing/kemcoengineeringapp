# Overflow Fitting Assembly Part Numbers Reference

## Overview
This document lists all part number assignments for Overflow Fitting Assembly:1 based on:
- Flange Size (2", 2.5", 3", 4", 6", 8", 10", 12")
- Material: SS304 or SS316
- Overflow Type: "Short tank overflow" or "Standard tank overflow"
- SS316_FLANGES flag (True/False) - for SS316 only
- Food Grade (currently commented out in code)

---

## Decision Tree

```
Is MATERIAL = "SS304"?
├─ YES
│  ├─ Is OverflowType = "Short tank overflow"?
│  │  ├─ YES → Use 1023429-XX series
│  │  └─ NO (Standard) → Use 1025543-XX series
│  └─ Is FoodApplication = True? (Currently disabled)
│     └─ YES → Use 1051520-XX (Short) or 1051521-XX (Standard)
│
└─ NO (SS316)
   ├─ Is OverflowType = "Short tank overflow"?
   │  ├─ YES
   │  │  ├─ Is SS316_FLANGES = True?
   │  │  │  ├─ YES → Use 1036402-XXSS series
   │  │  │  └─ NO → Use 1036402-XX series
   │  │  └─ Is FoodApplication = True? (Currently disabled)
   │  │     └─ YES → Use 1051838-XX series
   │  └─ NO (Standard)
   │     ├─ Is SS316_FLANGES = True?
   │     │  ├─ YES → Use 1033918-XXSS series
   │     │  └─ NO → Use 1033918-XX series
   │     └─ Is FoodApplication = True? (Currently disabled)
   │        └─ YES → Use 1051839-XX series
```

## Notes

- **Note**: Code has `iLogicVb.UpdateWhenDone = True` at the end (applies to all conditions)
- **Note**: SS304 does NOT use SS316_FLANGES flag (only SS316 uses it)
- **Note**: All food grade options are currently commented out in the code
- **Note**: Description differs between "Short" (VENT/OVERFLOW FITTING) and "Standard" (FLANGE FITTING)
